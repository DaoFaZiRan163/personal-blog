## 项目部署

这是一个轻量级静态个人博客，可以直接部署到 Vercel 或其他静态托管平台。

### 本地预览

```bash
# 使用任意静态服务器
npx serve .
# 或
python -m http.server 8000
```

### 部署到 Vercel

1. 创建 GitHub 仓库并推送项目
2. 在 Vercel 新建项目，选择该 GitHub 仓库
3. Framework Preset 选择 `Other`，Build Command 留空，Output Directory 留空或填 `.`
4. 部署完成后，绑定自定义域名

### 自定义内容

- `index.html` - 个人信息和首页内容
- `about.html` - 详细经历介绍
- `projects.html` - 项目案例展示
- `blog.html` - 博客文章列表
- `styles.css` - 颜色主题和排版样式
- `posts/` 目录 - 博客文章内容
